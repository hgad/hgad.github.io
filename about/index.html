About

Me

My name is Haitham Gad. I've been developing software for the past 7 years. My primary interest is in programming languages. I like to compare and contrast the most prominent features of different languages and programming paradigms. I've written code in many languages including C, C++, Haskell, Lisp, Python, Perl and Prolog. I'm also interested in some areas of mathematics and theoretical computer science including mathematical logic, formal methods and type theory.

I'm originally from Cairo, Egypt and I currently live in Allison Park, PA. My formal education is in Electrical Engineering (earned a B.Sc. EE from Cairo University in 2006). I used to work for an EDA company when I was in Cairo, until I moved to the company's headquarters in Oregon back in 2009. I now work for yet another EDA company in their Pittsburgh local office.

Colophon

This website is built with Hakyll and hosted on GitHub Pages. It uses the HTML5 Boilerpate as a template, and the style sheets are generated using Sass. Although I didn't utilize the full palette, most of the colors used in this website are from the Solarized colorshceme. Blog posts are written in Markdown and converted to HTML using Pandoc.
